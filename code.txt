3. Core Calculation Engine

Implement a shared calculation engine.

Formula

For each course:

weighted_point = course_unit × grade_point

Then:

GPA = sum(weighted_points for semester) / sum(units counted in semester)

CGPA = sum(all cumulative weighted_points) / sum(all cumulative counted units)

Support both:

direct grade entry
mark-to-grade conversion
Grade mapping

Use this mapping:

70 and above = A = 5.0
60–69 = B = 4.0
50–59 = C = 3.0
45–49 = D = 2.0
40–44 = E = 1.0
Below 40 = F = 0.0

Add utility functions for:

markToGrade(mark)
gradeToPoint(grade)
calculateWeightedPoint(unit, point)
calculateGPA(courses)
calculateCGPA(allCourses)

Round displayed GPA/CGPA to 2 decimal places, but keep internal precision safe.
